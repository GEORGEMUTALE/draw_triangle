# Triangle Drawing Tool Documentation
This is a GitHub Pages site for the Triangle Drawing Tool.
## Features
- Draw customizable triangles with any character.
- Supports command-line arguments for height and character.
## Installation
1. Download the binary for your platform from the [Releases](../../releases) section.
2. Run the binary from your terminal.
## Usage
```bash
./triangle-draw --height 5 --character '#'
```
